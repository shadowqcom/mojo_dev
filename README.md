# Mojo_dev  
Mojo开发者社区  
Mojo Developer Community
  

## 愿景  

"用Mojo写一个Mojo社区！"  

  
## 我们的价值观  
我们坚信以下价值观将指导我们的社区发展：  

- 开放和包容

- 学习和共享

- 创新和实践
  

## 如何参与  
如果您对 Mojo 已有较深入的了解，并具备开发能力，您可以贡献代码和文档给 Mojo Developer Community。  
这将使您的工作被更多人使用和受益，并为社区的发展做出贡献。
